## 4. 예상 효과

### 4.1 데이터 수집량
- **현재**: 13개 뉴스
- **예상 (3개월 기준)**: 500-1,000개 뉴스
- **증가율**: 약 40-80배

### 4.2 예측 정확도 향상
1. **지역별 세부화**: 동별 예측 차별화 가능
2. **교통망 다각화**: GTX 외 다른 교통 인프라 영향 반영
3. **가격 직접 반영**: 시장 심리 지표 강화
4. **정책 민감도**: 규제 변화에 대한 시장 반응 포착
5. **환경 요인**: 학군, 녹지 등 주거 선호도 반영

### 4.3 피처 엔지니어링
- **기존 피처**: 4개 (GTX, 재개발, 공급, 정책)
- **확장 피처**: 8개 (+ 지역, 사업, 경제, 사회)
- **피처 증가율**: 100%

## 6. 주의사항

### 6.1 API 호출 비용
- **키워드 증가**: 4개 → 15개 (3.75배)
- **예상 API 호출**: 15개 쿼리 × 50개 결과 = 750회/실행
- **권장 실행 주기**: 주 1회 (비용 절감)

### 6.2 데이터 품질 관리
- **노이즈 필터링**: 너무 일반적인 키워드는 관련 없는 뉴스 수집 가능
- **중복 제거**: 동일 뉴스가 여러 키워드로 수집될 수 있음
- **품질 검증**: 수집 후 관련성 점수 확인 필요

### 6.3 스토리지
- **현재**: 13개 뉴스 × 2KB = 26KB
- **예상**: 1,000개 뉴스 × 2KB = 2MB
- **Supabase Free 플랜**: 500MB (충분)
